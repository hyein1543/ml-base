# Background Maker

Story-to-background module that generates a **JRPG pixel-art background** (To the Moon / RPG Maker XP vibe)
with optional **reference-photo grounded objects**, then exports:

- `full background` (non-transparent)
- `object-only` (white background)
- `object cutout` (transparent PNG alpha via rembg)
- `background-only` (core objects removed & filled)

## Install
```bash
pip install -r requirements.txt
